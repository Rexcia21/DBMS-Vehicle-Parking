# DBMS Vehicle Parking System

## 📌 Project Description

This project is a **Vehicle Parking Management System** developed using **PHP and MySQL**.
It helps in managing vehicle entry, exit, and automatic fee calculation efficiently.

---

## ⚙️ Technologies Used

* 💻 PHP (Backend)
* 🗄️ MySQL (Database)
* 🌐 HTML, CSS (Frontend)
* 🛠️ XAMPP (Local Server)

---

## 🚀 Features

* Add vehicle entry with time
* Record vehicle exit
* Automatic parking fee calculation
* Store all data in database
* Simple and user-friendly interface

---

## 📁 Project Structure

```
vehicle-parking/
│
├── index.php
├── add_vehicle.php
├── exit_vehicle.php
├── config.php
├── db.sql
```

---

## ▶️ How to Run

1. Install XAMPP
2. Move project to:

   ```
   C:\xampp\htdocs\
   ```
3. Start Apache & MySQL
4. Open phpMyAdmin and import `db.sql`
5. Run in browser:

   ```
   http://localhost/vehicle-parking
   ```

---

## 📊 Database Details
